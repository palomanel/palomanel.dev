# palomanel.dev

[![pre-commit](https://img.shields.io/badge/pre--commit-enabled-brightgreen?logo=pre-commit)](https://github.com/pre-commit/pre-commit)
[![Conventional Commits](https://img.shields.io/badge/Conventional%20Commits-1.0.0-%23FE5196?logo=conventionalcommits&logoColor=white)](https://conventionalcommits.org)

Personal development blog hosted on
[GitHub Pages](https://docs.github.com/en/pages/),
and created using [Jekyll](https://jekyllrb.com/).

## Usage

Jump to [palomanel.dev](https://www.palomanel.dev/) to read the blog.
The [first post is about bootstraping](https://www.palomanel.dev/jekyll/update/2026/01/07/bootstraping-a-blog.html)
and provides more info about the project's structure.

Anyone is very welcome to clone the source and experiment, just have a
look at the [license information](LICENSE.md) beforehand.

## Contributing

This is a personal blog, so it's not expected for others to contribute
posts for now.

Comments, bug reports, or simply improvement suggestions are very welcome.
Feel free to submit an issue!

if you got inspiration from the code here to kickstart your own blog
and want to contribute back with an enhancement, you can submit
a pull request.
